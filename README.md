# Automatic_Web_Scrapper

# How to run?
### STEPS:


```bash
conda create -n mlproj python=3.11.5 -y 
```

```bash
conda activate mlproj
```


```bash
pip install -r requirements.txt
```

```bash
python app.py
```

```bash
Now open up your local host 0.0.0.0:8080
```
